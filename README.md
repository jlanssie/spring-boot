# Spring Boot Demo

Spring Boot Demo is a Test-Driven Design Demo app that showcases basic Spring Boot functionalities. 

## Installation

Use Gradle to install Spring Boot Demo.

```bash
./gradlew clean build
```

## Usage

Use Gradle to run the code.

```bash
./gradlew clean bootRun
```
